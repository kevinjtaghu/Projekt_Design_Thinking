# Projekt: Design Thinking für IT
## Home Office Balance Assistant - Prototyp

Dies ist ein UI/UX-Prototyp zur Demonstration meines Konzepts und nicht als fertige Anwendung gedacht.

### Funktionen

- Starten der Arbeitszeit mit integriertem Timer
- Fokus-Modus mit Hinweisen zur Reduktion von Ablenkungen
- Individuelle Einstellungen (Arbeitszeit & Pausen)
- Dynamische Tagesplanung basierend auf Nutzerangaben
- Feierabend-Screen mit Zusammenfassung der geleisteten Arbeit
  
### Ziel

Verbesserung von Fokus, Produktivität und Work-Life-Balance im Home Office.

### Technischer Stand

- Reiner Frontend-Prototyp (HTML, CSS, JavaScript)
- Keine Backend-Anbindung
- Daten werden nicht persistent gespeichert
